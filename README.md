# Shivam Shukla — Portfolio 🚀

My personal developer portfolio built with **React**, **TypeScript**, **GSAP**, and **Three.js**.

> Backend & AI Developer | 2026 CSE Graduate | Java • Spring Boot • LLM Integration

---

## ✨ Features

- **Two-Column Hero Section** — Profile photo on the left, bio + CTAs + social links on the right
- **Smooth Scroll Animations** — Powered by GSAP ScrollTrigger & ScrollSmoother
- **3D Tech Stack Visualizer** — Interactive Three.js canvas with floating tech icons
- **Timeline Sections** — Engineering Journey & Career milestones with animated scroll-driven timelines
- **Project Showcase** — Cards with modal overlays for detailed project breakdowns
- **Certifications & Current Focus** — Highlighting ongoing learning and specializations
- **Fully Responsive** — Optimized for desktop, tablet, and mobile viewports
- **Dark Theme** — Sleek `#0a0a0a` background with `#00d4a0` teal accent

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/shivam-shukla888/shivam-portfolio.git
cd shivam-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The site will be available at `http://localhost:5173/`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
